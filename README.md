# Mastering the Zebra Wireless Display API

This repository holds the sample code linked to the blog post about the Zebra Wireless Display API.
Such API allow developers to embed into their apps the wireless connection feature that is otherwise available through a separate APK [at this link](https://www.zebra.com/us/en/support-downloads/software/mobile-computer-software/wireless-workstation-connect.html?downloadId=a773e381-08bd-4062-9265-ceedaaca50e8).
